Terms Definitions
machine language This is base computer language made using 0s and 1s
debugging locating and correcting errors
assembler converts mnemonic expressions into machine language
assembly language this is a mnemonic system for representing programs
translator this is a program that transates programs expressed in high-level primitives into machine-language programs
compiler this is a translation program that compiles several machine instructions into short sequences
interpreter this is an alternative to a translator that execute instructions as they are translated
cross-platform software this is software that is independent of an operating system's design
natural language this is a language such as English, German, or Latin
formal language this would be a language such as a programming language
machine independence this means that a program can be run on multiple machines; it isn't just made for a specific one
high-level programming language this is a programming language with a strong abstraction from the details of the computer
object-oriented programming (OOP) This is a programming paradigm based on the concept of "objects" which contain data and code
scripting language this is a programming language that is used to manitplate, customize, and automate the facilities of an existing system
